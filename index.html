🛠️ Anleitung für GitHub

1. Datei speichern

Speichere diese Datei als index.html.

2. Repository erstellen

Erstelle auf GitHub ein neues Repository, z. B. finanzberater-lernen.

3. Datei hochladen

Lade index.html ins Hauptverzeichnis hoch.

4. GitHub Pages aktivieren

Unter Settings → Pages wähle Deploy from branch, Branch main, Ordner /root.

5. Fragen ergänzen

Neue Fragen fügst du im JavaScript im Objekt APP_DATA unter dem jeweiligen Thema hinzu.

6. Quellen ergänzen

Pro Frage kannst du source.document, source.section und source.note pflegen.
